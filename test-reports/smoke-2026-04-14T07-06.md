### Root Cause

The MCP server that provides `browser_navigate`, `browser_snapshot`, `browser_click`, `browser_fill_form`, and related tools is defined in `.github/mcp-config.json`:

```json
{
  "mcpServers": {
    "playwright": {
      "command": "npx",
      "args": ["-y", "@playwright/mcp", "--headless"]
    }
  }
}
```

This config is only loaded when Claude Code is started with the `--mcp-config` flag:

```

```
