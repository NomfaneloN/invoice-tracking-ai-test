## Recommendations

1. **Fix Mode badge label** (TC-All issue #1) — Display "Latest" when mode is actually latest. Currently always shows "Live" and confuses users.
2. **Fix sidebar flyout z-index** (issue #2) — The hover flyout overlay should not intercept clicks on main content when sidebar is collapsed.
3. **Reconcile spelling** (issue #3) — Align labels: "Capture Filing" (BAS) vs "Capture Filling" (LOGIS). Pick one canonical term.
4. **Continue using real BAS register templates** for TC-08 — The small 1-row E2E template works when the date columns (H10/O10/P10/Q10) are correctly set to today's Excel serial.
