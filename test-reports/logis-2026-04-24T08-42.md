## Recommendations

1. **TC-12 fix priority** — Capture Filling Submit is a complete workflow blocker. Should be top of queue when dev returns.
2. **Fix TC-05 dialog logic** — If "Verification is complete" is checked, Submit should NOT trigger the "Send for business related query" dialog. Also fix the typo ("relatd" → "related") noted as known bug.
3. **Fix TC-06 Total Amount calculation** — shows 3× the invoice sum on Capture and Link page.
4. **Improve TC-04 combobox selection feedback** — either the wrapper element should commit the selection, or Submit should re-validate that the combobox has a real id value (not just display text) before enabling.
