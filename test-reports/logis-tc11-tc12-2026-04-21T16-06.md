## Workflow Order Discrepancy (important finding)

The test plan previously listed TC-11 as `Capture Filling` and TC-12 as `Attach Payment Stub`, implying Capture Filling happens first. In reality, the system enforces:

**TC-11 Attach Payment Stub → TC-12 Capture Filing** (stub confirms payment, which sets status to `Paid`, which then unlocks Capture Filing for Internal Control).

The test plan has been updated this session to reflect the actual order.
