## Recommendations

1. **UI — Mode badge:** Fix the "Live" badge in the header to correctly reflect `CONFIGURATION_ITEM_MODE = "latest"` as "Latest" or similar. The current mismatch causes confusion about which mode is active.
2. **My Items vs Inbox:** Clarify or unify the distinction between "My Items" and "Inbox". If Capture Filing steps only appear in Inbox, ensure documentation and user guides reflect this, or make them visible in both views.
3. **Payment Number display:** Investigate the "Payment Number" field on the Capture Filing form. If it is intended to show the BAS PAYMENT NUMBER from the stub, it should reflect `5010`, not an unrelated code (`Yapp679`). If it is a different field, the label should be more descriptive.
